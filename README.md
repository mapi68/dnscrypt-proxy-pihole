## Overview
<b>Preconfigured deb package for every Raspberry Pi and Pi-hole to use only best dnscrypt , DNS-over-HTTPS and no log servers</b><br>

## Installation
```bash
curl -sSfL https://raw.githubusercontent.com/mapi68/dnscrypt-proxy-pihole/master/dnscrypt-proxy-pihole-install | bash
```
