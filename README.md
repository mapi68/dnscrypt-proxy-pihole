## Overview
<b>Preconfigured deb for every Raspberry Pi and Pi-hole to use encrypted DNS transport with 1.1.1.1</b><br><br>

## Installation
```bash
wget https://github.com/mapi68/dnscrypt-proxy-pihole/raw/master/dnscrypt-proxy-pihole_2.0.44_armhf.deb
dpkg -i dnscrypt-proxy-pihole_2.0.44_armhf.deb
```
dnscrypt-proxy-pihole NO NEED CONFIGURATION!<br><br>
